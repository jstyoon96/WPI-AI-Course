## How To Run A Lab

1. Open the notebook for the assigned lab.
2. Click the Colab badge at the top of the notebook.
3. Run the setup cell first.
4. Work through each section in order.
5. Complete every visible `TODO`.

Use a GPU runtime only when a lab explicitly asks for one.

## Grading And Submission

Each lab is graded out of `100 pts`.

- Completed notebook with generated plots, tables, and required outputs:
  `60 pts`
- Word response document with the lab questions: `40 pts`

Use the filename shown inside each notebook for the Word response document.

## Data

This repository does not store raw datasets. Lab notebooks use public or
library-provided data when possible.

Shared data loading helpers live in:

```text
wpi_ai_bootcamp/data/
```

These files contain loader code and source metadata, not large data files.

## Attribution

Datasets, images, libraries, and external references used in a lab are listed
in that notebook's attribution section.
