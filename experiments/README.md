# Experiments

Experiment outputs belong under `experiments/EXP_ID/`.

Each completed experiment should save:

- `config.yaml`
- `resolved_config.yaml`
- `git_commit.txt` when the repository is initialized
- `dataset_manifest.json`
- `train.log`
- `metrics_summary.json`
- `metrics_by_group.csv`
- `failure_report.json`
- `audit_report.json`
- `next_experiment_proposal.md`
- `artifacts/`

Do not claim scientific improvement unless the relevant metric improved under the defined evaluation protocol.
