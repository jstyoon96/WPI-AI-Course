# Attribution And License

## Course Material Policy
The WPI AI Bootcamp labs should be original student-facing material written for
this course. External references are allowed when they are cited clearly and
their licenses permit the intended use.

## MIT Reference
The MIT Introduction to Deep Learning repository is used as a structural
reference for organizing public lab materials. Do not copy MIT notebooks,
images, slide text, or completed answers into this repository without an
explicit license review.

Reference: <https://github.com/MITDeepLearning/introtodeeplearning>

## Required Attribution
For external material, record:

- Source name
- URL
- License, if available
- What was reused or adapted
- Date accessed

## Local-Only Materials
Instructor-only slides, completed answers, review notes, and raw drafts are not
published to GitHub.
